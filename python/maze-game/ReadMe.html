Zade's Maze

Code & version history is on GitHub.

Controls

Movement
W,A,S, D or Arrow Keys or Numpad 8,4,6,2 (Often have arrows on them)
Restarting
Space or R or Enter (Normal keypad or Numpad)

Gameplay

Completing levels

To complete a level, find your way through the maze to the level-end marker (pictured below).

Win the game by completing all the levels.

Powerups

There are powerups which can enchance certain attributes of the player. For example, the speed powerup (pictured below) applies a small multiplier to plaer acceleration.

Ghost walls

There are some walls that the player is able to walk through (if the player manages to discover them), which can be used as shorcuts to get lower times.

Ghost walls disapear after the player has walked through them.
